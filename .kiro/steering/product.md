# Product Overview

This is a monorepo containing a Tauri-based desktop application, with Python sidecars for enhanced functionality. The project leverages modern web technologies while utilizing Python's ecosystem for backend processing and AI/ML capabilities.

## Applications

### Desktop App (Tauri)
- Cross-platform native desktop application
- Simple React frontend with Tauri integration
- Python sidecars for specialized processing tasks
- Direct system integration capabilities

### Tauri React App (TanStack Start)
- Web application with full TanStack ecosystem which Tauri will leverage
- Advanced routing, state management, and data fetching
- Optimized for Tauri desktop application
- Rich interactive features

## Key Features
- Cross-platform desktop application built with Tauri
- Python sidecars for specialized processing tasks
- Shared TypeScript types across applications
- Cloud deployment capabilities via AWS CDK
- Automated CI/CD pipeline through GitHub Actions

## Environment
- Uses HuggingFace integration for AI/ML capabilities
- Audio processing functionality (based on audio assets)
- Hybrid architecture combining web technologies with native performance
- Monorepo structure for code sharing and consistency
