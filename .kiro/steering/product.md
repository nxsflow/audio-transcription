# Product Overview

This is a Tauri-based desktop application with Python sidecars for enhanced functionality. The application leverages modern web technologies for the frontend while utilizing Python's ecosystem for backend processing and AI/ML capabilities.

## Key Features
- Cross-platform desktop application built with Tauri
- Python sidecars for specialized processing tasks
- Modern React-based UI with TanStack libraries
- Cloud deployment capabilities via AWS CDK
- Automated CI/CD pipeline through GitHub Actions

## Environment
- Uses HuggingFace integration for AI/ML capabilities
- Audio processing functionality (based on audio assets)
- Hybrid architecture combining web technologies with native performance
